# Secure Multi-Cloud AI Fraud Detection Platform

## Project Background
While working in banking, I observed how financial fraud increasingly targets digital transaction systems. I designed this project to simulate a secure AI-powered fraud detection platform that protects sensitive financial data across multi-cloud environments.

## Project Goal
Build a Zero-Trust secured AI fraud detection pipeline using AWS Bedrock while implementing financial compliance controls such as PCI-DSS and GDPR.

## Technologies
- AWS Bedrock
- API Gateway
- AWS Lambda
- IAM Zero Trust Design
- AWS KMS Encryption
- AWS WAF
- GuardDuty
- CloudTrail
- Security Hub
- Azure Entra ID (Multi-Cloud Identity)
